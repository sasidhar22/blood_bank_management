# 🩸 Blood Bank Management System

A front-end web project built during **2nd Semester (2020)** — a Blood Bank Management System that allows users to register, donate blood, check blood stats, and more.

## 🌐 Live Site

[![Live Site](https://img.shields.io/badge/Live%20Site-View%20Here-red?style=for-the-badge)](https://sasidhar22.github.io/blood_bank_management/)

## 📋 About

This project simulates a Blood Bank Management portal with multiple pages for user login, signup, donation, and blood availability statistics. Built as a 2nd semester academic project using pure HTML and CSS.

## 🚀 Features

- 🏠 **Home Page** — Introduction and navigation
- 🔐 **Login & Signup** — User authentication pages
- 💉 **Donate Blood** — Blood donation form
- 📊 **Blood Stats** — View blood availability
- 🙋 **About Us** — Information about the organization
- 💬 **Feedback** — User feedback form

## 🛠️ Tech Stack

- HTML5
- CSS3

## 🏃 Getting Started

To run locally:

```bash
git clone https://github.com/sasidhar22/blood_bank_management.git
cd blood_bank_management
open index.html
```

Or just open `index.html` in your browser.
